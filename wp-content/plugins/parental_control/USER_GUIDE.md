# راهنمای کاربری پلاگین کنترل والدین
# Parental Control Plugin - User Guide

---

## 📱 برای والدین / For Parents

### شروع کار / Getting Started

#### ۱. ورود به حساب کاربری / Step 1: Login to Your Account
```
وارد سایت شوید با نام کاربری و رمز والدین خود
Login to the website with your parent username and password
```

#### ۲. رفتن به داشبورد والدین / Step 2: Go to Parent Dashboard
```
بعد از ورود خودکار به صفحه پروفایل والدین می‌روید
After login you will automatically go to parent profile page
```

#### ۳. ایجاد حساب فرزند / Step 3: Add a Child Account
```
روی دکمه "افزودن فرزند" کلیک کنید
Click on "Add Child" button

سپس:
Then:
- نام کاربری فرزند را وارد کنید (مثل: بچه_۱)
  Enter username (example: child_1)
- رمز عبور انتخاب کنید (حداقل ۸ کاراکتر)
  Choose a password (at least 8 characters)
- کلیک کنید "ایجاد"
  Click "Create"
```

#### ۴. تنظیم اجازه‌های فرزند / Step 4: Set Child Permissions

**۳ قسمت اجازه وجود دارد:**
**There are 3 permission categories:**

##### الف) گروه سنی / Age Groups
```
انتخاب کنید فرزند شما حداقل چند سالشه:
Select the appropriate age range for your child:

🟢 ۳+  : برای بچه‌های ۳ سال و بالاتر
         For ages 3 and up

🟢 ۵+  : برای بچه‌های ۵ سال و بالاتر
         For ages 5 and up

🟡 ۱۳+ : برای نوجوانان ۱۳ سال و بالاتر
         For teenagers 13 and up

🔴 ۱۸+ : فقط برای بزرگسالان ۱۸ سال و بالاتر
         For adults 18 and up only
```

##### ب) نوع محتوا / Content Genres
```
انتخاب کنید چه نوع مطالب می‌خواهید فرزند ببیند:
Choose what types of content your child can see:

✓ آموزشی (Educational)
✓ علمی‌تخیلی (Science Fiction)
✓ کمدی (Comedy)
✓ ماجراجویی (Adventure)
✓ درام (Drama)
✓ مستند (Documentary)
✓ انیمیشن (Animation)
✓ ترسناک (Horror)
```

##### ج) هشدارهای محتوا / Content Warnings
```
انتخاب کنید کدام محتواهای خطرناک را مسدود کنید:
Choose which dangerous content to block:

⛔ خشونت (Violence)
⛔ زبان نامناسب (Inappropriate Language)
⛔ محتوای جنسی (Sexual Content)
⛔ مواد مخدر (Drug References)
⛔ صحنه‌های ترسناک (Scary/Intense Scenes)
```

#### ۵. ذخیره‌سازی / Step 5: Save
```
بعد از انتخاب، روی دکمه "ذخیره اجازه‌ها" کلیک کنید
Click "Save Permissions" button

پیام موفقیت نشان داده خواهد شد
Success message will appear
```

### مثال عملی / Practical Example

**سناریو: فرزندی ۸ ساله**
**Scenario: 8-year-old child**

```
✓ گروه سنی: 5+ (مناسب برای ۵ سال و بالاتر)
  Age Group: 5+ (suitable for age 5+)

✓ نوع محتوا: آموزشی، انیمیشن، ماجراجویی
  Genres: Educational, Animation, Adventure

⛔ هشدارها: خشونت، صحنه‌های ترسناک
  Block: Violence, Scary scenes

نتیجه: بچه فقط می‌تواند محتوایی را ببیند که:
Result: Child can only see content that:
- برای ۵ سال و بالاتر مناسب باشد
  Is suitable for 5+ years
- از نوع آموزشی، انیمیشن یا ماجراجویی باشد
  Is Educational, Animation, or Adventure
- خشونت و صحنه‌های ترسناک نداشته باشد
  Does NOT have violence or scary scenes
```

### تغییر اجازه‌ها / Changing Permissions

```
هر وقت بخواهید اجازه‌ها را تغییر دهید:
If you want to change permissions:

۱. فرزند را انتخاب کنید
   Select the child

۲. اجازه‌های جدید را انتخاب کنید
   Choose new permissions

۳. روی "ذخیره" کلیک کنید
   Click "Save"

تغییرات فوری بر روی محتوای فرزند اعمال می‌شود
Changes take effect immediately
```

### مدیریت رمز عبور / Managing Passwords

```
برای تغییر رمز عبور فرزند:
To change child's password:

۱. به داشبورد والدین بروید
   Go to parent dashboard

۲. روی نام فرزند کلیک کنید
   Click on child's name

۳. روی "تغییر رمز عبور" کلیک کنید
   Click "Change Password"

۴. رمز عبور جدید را وارد کنید
   Enter new password

۵. کلیک کنید "تغییر"
   Click "Change"
```

## 👶 برای بچه‌ها / For Children

### ورود و استفاده / Login and Use

```
۱. نام کاربری خود را وارد کنید
   Enter your username

۲. رمز عبوری که والدین داده‌اند را وارد کنید
   Enter password your parent gave you

۳. کلیک کنید "ورود"
   Click "Login"

۴. مطالب مجاز را بخوانید
   Read allowed content

✓ اگر یک مطلب برای شما مناسب نیست، نمی‌توانید آن را ببینید
  If content is not suitable for you, you won't be able to see it

✓ این برای حفاظت از شما است
  This is to protect you
```

### اگر محتوا دیده نمی‌شود / If You Can't See Content

```
اگر می‌خواهید محتوای خاصی را ببینید اما نمی‌توانید:

If you want to see certain content but can't:

۱. به والدین بگویید کدام محتوا می‌خواهید
   Tell your parent what content you want

۲. والدین می‌تواند اجازه‌های شما را تغییر دهند
   Your parent can change your permissions

۳. سپس می‌توانید آن محتوا را ببینید
   Then you can see that content

⚠️ والدین آن را انجام می‌دهند تا شما محفوظ باشید
   Your parent does this to keep you safe
```

## ⚙️ تنظیمات سایت / Site Settings

### دسترسی به تنظیمات / Accessing Settings

**فقط مدیران سایت می‌توانند این کار را کنند**
**Only website administrators can do this**

```
۱. وارد ناحیه مدیریت شوید
   Go to admin area

۲. به Settings > Parental Control برید
   Go to Settings > Parental Control

۳. تنظیمات را پیکربندی کنید
   Configure settings
```

### گزینه‌های تنظیمات / Setting Options

| تنظیم | توضیح | Setting | Description |
|------|-------|---------|-------------|
| **Max Children** | حداکثر فرزند برای هر والدین | Maximum children per parent | 3 (default) |
| **Parent Redirect** | صفحه‌ای که والدین بعد از ورود می‌روند | Where parents go after login | `/parent-profile/` |
| **Child Redirect** | صفحه‌ای که بچه‌ها بعد از ورود می‌روند | Where children go after login | `/` (home) |

## ❓ سؤالات متداول / FAQ

### س۱: اگر فرزندم رمز خود را فراموش کند؟
Q1: What if my child forgets their password?

**ج:** والدین می‌تواند از داشبورد خود رمز عبور فرزند را تغییر دهد
**A:** Parent can change child's password from parent dashboard

---

### س۲: آیا می‌تواند دو فرزند نام کاربری یکسان داشته باشند؟
Q2: Can two children have the same username?

**ج:** خیر، هر فرزند باید نام کاربری منحصر به‌فرد داشته باشد
**A:** No, each child must have a unique username

---

### س۳: اگر اجازه‌ها را اشتباه تنظیم کردم؟
Q3: What if I set permissions incorrectly?

**ج:** می‌تونید هر وقت اجازه‌ها را تغییر دهید. تغییرات فوری اعمال می‌شود
**A:** You can change them anytime. Changes take effect immediately

---

### س۴: چقدر طول می‌کشد تغییرات اعمال شود؟
Q4: How long does it take for changes to apply?

**ج:** فوری - هنگام ذخیره‌سازی
**A:** Instantly - when you save

---

### س۵: آیا بچه می‌تواند اجازه‌های خود را تغییر دهد؟
Q5: Can child change their own permissions?

**ج:** خیر، فقط والدین می‌توانند اجازه‌ها را تغییر دهند
**A:** No, only parent can change permissions
